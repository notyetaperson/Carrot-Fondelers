# 🥕 Carrot Fondelers

**A Silly RPG in Three.js**

A chaotic, humorous 3D action RPG where you run around fondeling carrots, fighting pigs, negotiating with wildlife, and causing general absurdity.

---

## ✨ Features

- Full 3D open world built with Three.js
- Combat system with melee, guns, and special abilities
- Negotiation / "Parley" mechanics with pigs and other creatures
- Potion drinking, shockwaves, phase dashes
- Progression system with ranks and medals
- Emotes, mobile controls, and dev console
- Dynamic UI with damage effects, health, and kill counter
- Procedural skybox and environment

---

## 🎮 How to Play

### Easiest Way (Windows)
1. Download or clone the repository
2. Double-click **`Run.bat`**
3. Open your browser and go to **http://localhost:8080**

### Alternative (Python)
```bash
python -m http.server 8080
